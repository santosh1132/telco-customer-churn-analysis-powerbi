# Telco Customer Churn Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes customer churn patterns and provides insights into customer retention.

## Technologies Used
- Power BI
- DAX
- Excel

## Key Insights
- Total Customers
- Churn Customers
- Churn Rate
- Customer Tenure Analysis
- Internet Service Analysis
- Contract Type Analysis
- Payment Method Analysis
